Class StaticMeshActor

module

LifeCoreObject

instance

StaticMesh

description

StaticMeshActor is an instance of a StaticMesh in the world. Static meshes are geometry that do not animate or otherwise deform, and are more efficient to render than other types of geometry. Static meshes dragged into the level from the Content Browser are automatically converted to StaticMeshActors.
