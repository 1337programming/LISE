# LISE
Life In Shambles Engine

## Requirements
1. Node
2. Git


## Setup 
Clone:
`git clone https://github.com/1337programming/LISE.git`

Install:
`npm install` or `yarn install`

Run:
`npm start`

Build:
`npm run build`

## Tasks

- Serve: `npm start`
- Build: `npm run build`
- Test: (Comming Soon)
- Lint: `npm run lint`
